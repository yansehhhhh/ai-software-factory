# 公司内部投票系统

一个基于 Vue 3 + Vite 的公司内部投票网页演示工程。

## 功能覆盖

- 员工/管理员邮箱或工号加密码登录演示
- 管理员创建和管理投票
- 匿名/实名投票模式
- 单题/多题投票
- 单选、多选、评分、文本反馈题型
- 投票创建后立即开始，仅设置结束时间
- 所有员工可参与每个投票
- 员工提交后不可修改
- 投票结束后查看图表化统计结果
- Excel/CSV 员工账号导入交互演示
- 响应式布局，支持桌面端和移动端

## 运行方式

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
