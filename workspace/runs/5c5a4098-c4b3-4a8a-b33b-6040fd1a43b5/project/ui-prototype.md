# 静态 HTML UI 原型说明

> 原型文件：`prototype/index.html`  
> 打开方式：直接用浏览器打开，无需构建、无需服务端、无需数据库  
> 项目：简单个人博客静态展示

---

## 1. 原型目标

本原型用于验证“简单个人博客”的信息结构、页面布局、视觉风格和基础交互。它不是 Vue/React 工程，也不依赖任何后端服务，所有 HTML、CSS 和 JavaScript 均集中在 `prototype/index.html` 内，便于直接预览和后续移植。

原型满足以下需求：

- 不需要登录。
- 不需要数据库。
- 静态展示博客内容。
- 包含首页、文章列表、文章详情。
- 使用内置示例文章。
- 风格为温暖文艺风，偏生活记录。
- 首页展示博客标题、简介、精选文章。

| Hash | 页面 |

## 5. 交互说明

### 5.1 页面切换

原型通过 `window.location.hash` 进行页面切换：

- 点击“首页”：切换到 `#home`
- 点击“文章”：切换到 `#posts`
- 点击文章卡片：切换到 `#post/{id}`

### 5.2 文章详情加载

文章详情根据 Hash 中的文章 ID 从内置 `posts` 数组查找：

```js
const post = posts.find(item => item.id === id);
```

找到文章则渲染详情页；未找到则展示 Not Found 视图。

### 5.3 精选文章筛选

首页精选文章通过字段筛选：

```js
posts.filter(post => post.featured)
```

### 5.4 文章排序

文章列表按日期倒序：

```js
[...posts].sort((a, b) => b.date.localeCompare(a.date))
```

### 5.5 键盘与可访问性交互

- 提供“跳到主要内容”链接。
- 顶部导航和文章卡片均可通过 Tab 聚焦。
- `:focus-visible` 提供明显焦点样式。
- 视图切换后通过 `aria-live` 向屏幕阅读器提示当前状态。
- 页面使用 `header`、`nav`、`main`、`section`、`article`、`footer` 等语义标签。

## 6. 内置示例文章

原型内置 4 篇文章：

| ID | 标题 | 分类 | 精选 |
|---|---|---|---:|
| `spring-walk` | 春日午后的散步 | 生活记录 | 是 |
| `weekend-book` | 一本书陪我度过周末 | 阅读随笔 | 是 |

| `travel-moments` | 旅行回来后的几个瞬间 | 旅行手记 | 否 |

文章数据结构：

```js
{
  id: "spring-walk",
  title: "春日午后的散步",
  date: "2026-04-12",
  category: "生活记录",
  featured: true,
  excerpt: "阳光落在街角的小店门口，风里有刚烤好的面包香。",
  content: ["正文段落一", "正文段落二"],

```

## 7. 视觉与设计系统应用

原型应用了以下设计系统规则：

- 页面背景：暖米白渐变。
- 主色：陶土棕。
- 辅助色：奶茶色、橄榄绿。
- 卡片：暖白背景、柔和边框、圆角和低透明阴影。
- 标题：大字号、紧凑行高、轻微负字距。
- 正文：较大字号、宽松行高、限制阅读宽度。
- 动效：仅使用轻微上浮、淡入，避免干扰阅读。
