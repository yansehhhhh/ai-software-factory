**标签筛选交互流程**：

```
1. 默认状态：显示 [标签筛选: 全部 ▼]
2. 用户点击下拉 → 展开标签列表
3. 用户选择 "Vue"
4. URL 变为 /posts/?tag=Vue
5. 筛选栏显示 [标签筛选: Vue ▼]（高亮激活态）
6. 文章列表只显示含 "Vue" 标签的文章
7. 分页按筛选结果重新计算

清除筛选：
- 点击筛选栏的 [✕] 或选择「全部」
- URL 回到 /posts/
```

**状态说明**：

| 状态 | 展示 |
|------|------|
| 有文章 + 无筛选 | 展示所有文章，分页 |
| 有文章 + 有筛选 | 展示该标签文章，分页 |
| 筛选无结果 | `"该标签下暂无文章"` + 返回全部链接 |
| 全部无文章 | `"暂无文章，敬请期待"` |
| 页码超出范围 | 显示最后一页，不出错 |

## 3. 关键组件说明

### 3.1 组件清单

| 组件名称 | 类型 | 复用范围 | 说明 |
|----------|------|----------|------|
| `SiteNav` | 全局 | 所有页面 | 顶部导航栏，含移动端汉堡菜单 |
| `SiteFooter` | 全局 | 所有页面 | 底部页脚 |
| `ArticleCard` | 业务 | 首页、列表页 | 文章卡片（标题、日期、摘要、标签） |
| `TagBadge` | 通用 | 全局 | 标签 badge，可点击 |
| `TagCloud` | 业务 | 首页、标签页 | 标签云/标签卡片网格 |
| `AuthorBio` | 业务 | 首页 | 作者简介区（头像、名称、简介、链接） |
| `Pagination` | 通用 | 列表页 | 分页组件 |
| `PostContent` | 业务 | 详情页 | Markdown 正文渲染区 |
| `PostMeta` | 业务 | 详情页 | 文章元信息（日期、作者、阅读时长） |
| `PostNav` | 业务 | 详情页 | 上一篇/下一篇导航 |
| `TagSelector` | 业务 | 列表页 | 标签筛选下拉组件 |

### 3.2 组件详细规约

#### TagBadge

```
┌─────────────────┐
│  TypeScript  ✕   │  ← 可关闭模式（筛选激活时）
└─────────────────┘

┌─────────────────┐
│  TypeScript      │  ← 默认模式（可点击）
└─────────────────┘
```

| 属性 | 类型 | 说明 |
|------|------|------|
| `label` | string | 标签文本 |
| `to` | string? | 点击跳转路由 |
| `closable` | boolean | 是否显示关闭按钮 |
| `size` | 'sm' \| 'md' | 尺寸 |

#### ArticleCard

| 属性 | 类型 | 说明 |
|------|------|------|
| `title` | string | 文章标题 |
| `date` | string | 发布日期 |
| `summary` | string | 摘要（≤150 字） |
| `tags` | string[] | 标签列表 |
| `to` | string | 文章详情链接 |

| 状态 | 说明 |
|------|------|
| 默认 | 正常显示 |
| hover | 阴影提升，标题颜色变化 |

#### Pagination

```
◀ 上一页   1   2   [3]   4   5   下一页 ▶
           ↑ 当前页高亮
```

| 属性 | 类型 | 说明 |
|------|------|------|
| `current` | number | 当前页码 |
| `total` | number | 总页数 |
| `basePath` | string | 分页路径前缀 |

#### SiteNav

| 状态 | 说明 |
|------|------|
| 桌面端 | 水平排列所有导航项 |
| 移动端（< 640px） | 折叠为汉堡图标 ☰，点击展开全屏菜单 |
| 菜单展开 | 覆盖全屏，显示导航项，点击遮罩或链接关闭 |
| 当前页面 | 对应导航项高亮/下划线 |

         │               │            │   │
         │    [点击卡片]  │            │   │
         │               ↓            │   │
         │          ┌──────────┐      │   │
         │          │ 文章详情  │      │   │
         │          └──────────┘      │   │
         │                            │   │

                    └──────────┘
```

### 4.2 文章列表筛选状态

```
初始状态
  │
  ├─ 用户选择标签 "Vue"
  │   └─→ 状态: 筛选激活
  │       URL: /posts/?tag=Vue
  │       显示: 仅 Vue 标签文章
  │       │
  │       ├─ 用户切换标签 "React"
  │       │   └─→ 状态: 筛选激活（新标签）
  │       │       URL: /posts/?tag=React
  │       │
  │       ├─ 用户点击清除 [✕]
  │       │   └─→ 状态: 回到初始
  │       │       URL: /posts/
  │       │
  │       └─ 筛选结果为空
  │           └─→ 状态: 空结果
  │               显示: "该标签下暂无文章"
  │               提供: 返回全部文章链接
  │
  └─ 用户点击分页
      └─→ 状态: 翻页
          URL: /posts/page/2
          文章列表更新
```

### 4.3 文章阅读流程

```
进入详情页
  │
  ├─ 文章存在 → 渲染正文
  │   ├─ Markdown → HTML
  │   ├─ 代码块 → Shiki 高亮
  │   ├─ 图片 → 懒加载 + alt 回退
  │   └─ 阅读进度条（P2）
  │
  ├─ 阅读完成
  │   ├─ 点击标签 → 跳转筛选列表
  │   └─ 点击上一篇/下一篇 → 跳转相邻文章
  │
  └─ 文章不存在 → 404
      └─ 提供返回首页链接
```

## 5. 交互细节说明

### 5.1 导航栏固定

- 顶部导航栏 `position: sticky; top: 0;`
- 背景从透明过渡到半透明毛玻璃（滚动 > 20px 时）
- z-index 高于内容区

### 5.2 文章卡片悬停

- `box-shadow` 从 `0 1px 3px` 过渡到 `0 4px 12px`
- 标题颜色从正文色变为主题色
- 过渡时间 `200ms ease`

### 5.3 标签 Badge 点击

- 路由跳转使用 Vue Router `<router-link>`
- 无页面刷新，SPA 内跳转
- 点击后有短暂的路由加载状态（顶部进度条）

### 5.4 移动端菜单

- 汉堡图标 ☰ 点击 → 展开全屏遮罩菜单
- 菜单项垂直排列，间距 16px
- 点击遮罩区域或导航链接 → 关闭菜单
- 关闭动画：菜单向上滑出

### 5.5 图片懒加载

- 文章详情中的图片使用 `loading="lazy"`
- 图片加载前显示骨架占位
- 图片加载失败显示 alt 文本

### 5.6 键盘导航（可访问性 P2）

- `Tab` 键在导航项、链接、按钮间切换
- `Enter` 键激活聚焦元素
- 焦点样式：轮廓线清晰可见

## 6. 响应式适配说明

### 6.1 断点策略

| 断点 | 宽度范围 | 布局模式 |
|------|----------|----------|
| 手机 | < 640px | 单列，汉堡菜单 |
| 平板 | 640px - 1024px | 主内容 + 可选侧边栏 |
| 桌面 | > 1024px | 主内容（max-width 960px 居中） |

### 6.2 各断点变化

| 元素 | 手机 (< 640px) | 平板 (640-1024px) | 桌面 (> 1024px) |
|------|---------------|--------------------|--------------------|
| 导航栏 | 汉堡菜单折叠 | 正常水平排列 | 正常水平排列 |
| 文章卡片 | 全宽，padding 16px | 全宽，padding 24px | 全宽，padding 32px |
| 字体大小 | 15px | 16px | 16px |
| 代码块 | 14px，横向滚动 | 15px | 15px |
| 标签云 | 2 列 | 3 列 | 4 列 |
| 分页按钮 | 紧凑间距 | 正常间距 | 正常间距 |
| 作者头像 | 100px | 140px | 180px |

### 6.3 安全区域

- 主内容区左右 padding: `max(16px, calc((100vw - 960px) / 2))`
- 移动端适配 safe-area-inset（iPhone 刘海屏）
