# 投票系统原型说明

**产品名称**: 公开投票管理系统
**文档版本**: v1.0
**创建日期**: 2026-04-27
**负责人**: Product Agent

---

## 1. 原型概述

本文档描述投票系统的页面结构、核心交互、状态流转和关键组件设计。系统分为用户端和管理端两部分，用户端提供投票参与功能，管理端提供活动管理功能。

---

#### 2.1.2 核心交互

**筛选交互**
- 点击状态标签（全部/进行中/已结束）筛选活动
- 在搜索框输入活动标题关键词搜索
- 筛选和搜索结果实时更新列表

**活动卡片交互**
- 点击活动卡片进入活动详情页
- 点击"立即参与"按钮进入活动详情页并提示投票
- 进行中的活动显示"立即参与"按钮
- 已结束的活动显示"查看结果"按钮

**登录交互**
- 点击顶部"登录"按钮，弹出登录弹窗
- 未登录用户点击"立即参与"时，先弹出登录弹窗

#### 2.1.3 关键组件

**活动卡片组件**
- 封面图片：200px x 150px，加载失败显示默认图片
- 活动标题：单行显示，超出部分省略号
- 状态标签：颜色区分（进行中-绿色，已结束-灰色）
- 参与人数：实时显示当前参与人数
- 操作按钮：根据状态显示不同文案

**筛选标签组件**
- 单选标签，选中状态显示高亮
- 点击切换选中状态
- 支持横向滚动（移动端）

**搜索框组件**
- 输入框带搜索图标
- 支持实时搜索（输入即搜索）
- 清空按钮快速清除搜索内容

#### 2.2.1 页面结构

```
┌─────────────────────────────────────────────┐
│                登录弹窗                      │
│  ┌───────────────────────────────────────┐ │
│  │ 关闭按钮 [×]                           │ │
│  │                                       │ │
│  │       手机号登录                       │ │
│  │                                       │ │
│  │  手机号输入框                          │ │
│  │  [请输入手机号]                        │ │
│  │                                       │ │
│  │  验证码输入框    获取验证码按钮        │ │
│  │  [请输入验证码]   [获取验证码]        │ │
│  │                                       │ │
│  │          [登录] 按钮                   │ │
│  │                                       │ │

```

#### 2.2.2 核心交互

**获取验证码交互**
- 输入手机号后点击"获取验证码"
- 手机号格式校验（11位数字，以1开头）
- 格式错误提示"请输入正确的手机号"
- 验证码发送成功后按钮变为倒计时（60秒）
- 倒计时期间按钮不可点击，显示"重新发送（剩余秒数）"

**登录交互**
- 输入验证码后点击"登录"
- 验证码校验（6位数字）
- 验证码错误提示"验证码错误，请重新输入"
- 验证码过期提示"验证码已过期，请重新获取"
- 登录成功关闭弹窗，更新页面登录状态

#### 2.2.3 关键组件

**手机号输入框**
- 11位数字输入限制
- 自动格式化显示（可选）
- 实时校验并提示错误

**验证码输入框**
- 6位数字输入限制
- 获取验证码后自动聚焦
- 输入完成后自动触发登录（可选）

**倒计时按钮**
- 发送成功后倒计时60秒
- 倒计时期间禁用状态
- 倒计时结束恢复"获取验证码"

#### 2.3.2 核心交互

**选项选择交互**
- 点击选项卡片或"选择"按钮选中该选项
- 选中选项显示选中状态（边框高亮、图标标记）
- 只能选择一个选项（单选）
- 选择后底部显示"已选择: [选项名称]"
- 选择后"确认投票"按钮变为可点击状态

**投票确认交互**
- 未登录：点击"确认投票"弹出登录弹窗
- 已登录未投票：点击"确认投票"弹出确认提示
- 已投票：提示"您已参与过投票，不能重复投票"

**投票成功交互**
- 投票成功后显示成功提示（Toast）
- 提示内容："投票成功，感谢您的参与！"
- 提示持续时间：3秒
- 提示消失后停留在活动详情页
- 底部按钮变为"已投票"状态（不可点击）

**活动状态交互**
- 活动未开始：投票按钮显示"活动未开始"，不可点击
- 活动进行中：正常投票流程
- 活动已结束：显示"查看投票结果"按钮，点击进入结果页

#### 2.3.3 关键组件

**选项卡片组件**
- 选项图片：150px x 150px（移动端），200px x 200px（PC端）
- 选项名称：最多30字符，超出显示省略号
- 选项描述：最多100字符，超出可展开查看
- 选中状态：边框蓝色高亮，右上角选中图标

**投票按钮组件**
- 未选择时：禁用状态，灰色，文案"请选择投票选项"
- 已选择未登录：文案"登录并投票"
- 已选择已登录：文案"确认投票"，高亮蓝色
- 已投票：禁用状态，灰色，文案"已投票"

**规则说明组件**
- 列表形式展示投票规则
- 可折叠展开（可选）

#### 2.4.2 核心交互

**排名展示交互**

- 第1名显示金色奖杯图标
- 第2名显示银色奖杯图标
- 第3名显示铜色奖杯图标

- 点击"柱状图"切换为柱状图展示

- 点击返回按钮返回活动详情页

- 票数显示：整数票数
- 占比显示：百分比，保留1位小数
- 进度条：根据占比显示长度，颜色渐变

**柱状图组件**
- X轴：选项名称
- Y轴：票数
- 柱子颜色：主色蓝色
- 悬停显示具体数值

**饼图组件**
- 圆形饼图展示占比
- 不同选项不同颜色
- 悬停显示选项名称和占比

## 4. 状态流转

### 4.1 用户登录状态流转

```
未登录状态
  ↓ 点击登录
弹出登录弹窗
  ↓ 输入手机号
  ↓ 获取验证码
验证码发送状态
  ↓ 输入验证码
  ↓ 点击登录
登录验证状态
  ├─ 验证成功 → 已登录状态
  ├─ 验证失败 → 返回登录弹窗
```

### 4.2 投票活动状态流转

```
创建活动（草稿）
  ↓ 点击发布
未开始状态（活动时间未到）
  ↓ 达到开始时间
进行中状态（可投票）
  ↓ 达到结束时间
已结束状态（可查看结果）
```

**状态说明**
- 草稿：活动创建中，未公开，可编辑删除
- 未开始：已发布但未到开始时间，不可投票
- 进行中：活动进行中，可投票，不可编辑选项
- 已结束：活动结束，可查看结果，不可编辑

### 4.3 用户投票状态流转

```
未登录状态
  ↓ 登录
已登录未投票状态
  ↓ 选择选项
  ↓ 点击确认投票
投票确认状态
  ↓ 投票成功
已投票状态
  ↓ 不允许再次投票
```

## 5. 交互细节说明

### 5.1 反馈提示

**成功提示**
- 使用 Toast 组件
- 背景色：绿色
- 显示位置：页面顶部中央
- 持续时间：3秒
- 自动消失，无需手动关闭

**错误提示**
- 使用 Toast 组件
- 背景色：红色
- 显示位置：页面顶部中央
- 持续时间：5秒或手动关闭
- 包含具体错误信息

**确认提示**
- 使用弹窗组件
- 标题：操作确认
- 内容：具体确认内容
- 按钮：确认/取消
- 确认后执行操作，取消后关闭弹窗

### 5.2 加载状态

**页面加载**
- 显示骨架屏
- 关键区域优先加载
- 加载失败显示错误状态

**按钮加载**
- 点击按钮后显示加载图标
- 按钮变为禁用状态
- 操作完成后恢复正常

**图片加载**
- 显示默认占位图
- 加载失败显示错误图标
- 支持重试加载

### 5.3 表单校验

**实时校验**
- 输入时实时校验格式
- 格式错误显示红色提示文字
- 格式正确隐藏提示文字

**提交校验**
- 点击提交时校验所有必填字段
- 缺失字段显示提示文字
- 校验失败不提交表单

### 5.4 空状态

**活动列表空状态**
- 显示空状态图标
- 文案："暂无投票活动"
- 按钮："刷新"或管理员创建活动

**投票记录空状态**
- 显示空状态图标
- 文案："您还没有参与过投票"
- 按钮："去看看"
