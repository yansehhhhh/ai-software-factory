### 已确认的需求基线

- **类型**：个人博客（单作者，文章展示为主）
- **内容管理**：本地 Markdown 文件 → Git push → 自动构建部署
- **技术栈**：推荐 VitePress（Vue 3 驱动，契合 GitHub Pages + Markdown + Git 流程）
- **功能范围**：首页 + 文章列表 + 文章详情 + 标签/分类 + 关于页
- **部署**：GitHub Pages
- **不包含**：评论、搜索、用户系统、后台管理、RSS
