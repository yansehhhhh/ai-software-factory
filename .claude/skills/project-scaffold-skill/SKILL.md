# Project Scaffold Skill

用于指导 Claude Code 生成符合本仓库架构的新项目脚手架和模板产物。

重点：
- 生成内容应贴合现有 monorepo 结构
- 前端优先参考 `packages/templates/vue3-js-template`
- 后端优先参考 `packages/templates/springboot-template`
- 测试优先参考 `packages/templates/playwright-template`
- 演示产物优先落到 `generated/`

说明：
- 这是作者工具层 skill，不等同于平台运行时 `packages/skills/`。

参考资料见 `skills/project-scaffold-skill/`、`packages/templates/` 与 `generated/`。
