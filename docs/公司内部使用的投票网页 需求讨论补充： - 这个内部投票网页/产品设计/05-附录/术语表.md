# 公司内部投票网页 - 附录

## 1. 术语表

| 术语 | 定义 |
|---|---|
| 投票活动 | 管理员创建的一次内部投票或意见收集活动，可包含单题或多题。 |
| 题目 | 投票活动中的一个问题，支持单选、多选、评分题、文本反馈。 |
| 单题投票 | 一个投票活动只包含一道题目。 |
| 多题投票 | 一个投票活动包含多道题目。 |
| 单选题 | 员工只能从多个选项中选择一个答案的题型。 |
| 多选题 | 员工可以从多个选项中选择多个答案的题型。 |
| 评分题 | 员工在指定评分范围内选择一个分值的题型。 |
| 文本反馈 | 员工以文本方式填写意见或补充说明的题型。 |
| 实名投票 | 管理员可查看投票人及其答案明细的投票模式。 |
| 匿名投票 | 系统不向任何结果查看者展示投票人与答案对应关系的投票模式。 |
| 进行中 | 投票创建后且未到结束时间、未被提前结束的状态。 |
| 已结束 | 投票到达结束时间或被管理员提前结束后的状态。 |
| 已提交 | 员工已对某投票完成提交，且不能再次提交或修改。 |
| 图表化统计 | 使用条形图、饼图、柱状图、指标卡等形式展示投票结果。 |
| Excel 批量导入 | 管理员通过上传 Excel 文件批量创建或更新员工账号。 |

## 3. WWAS 分析

WWAS：Who / What / Action / Success，用于描述谁在什么场景下完成什么动作以及成功标准。

| 编号 | Who | What | Action | Success |
|---|---|---|---|---|
| WWAS-01 | 员工 | 登录系统 | 输入邮箱/工号和密码并提交 | 登录成功，进入投票列表 |
| WWAS-02 | 员工 | 参与投票 | 打开进行中投票并填写答案 | 提交成功，系统标记已提交 |
| WWAS-03 | 员工 | 防止重复投票 | 已提交后再次进入投票 | 页面显示已提交，不能再次提交 |
| WWAS-04 | 员工 | 查看结果 | 投票结束后进入结果页 | 看到图表化统计结果 |
| WWAS-05 | 管理员 | 创建投票 | 填写基础信息、题目、实名/匿名、结束时间 | 投票创建成功并立即开始 |
| WWAS-06 | 管理员 | 配置题型 | 选择单选、多选、评分、文本反馈 | 题目保存成功并可供员工填写 |
| WWAS-07 | 管理员 | 导入账号 | 上传符合模板的 Excel | 员工账号创建或更新成功 |
| WWAS-08 | 管理员 | 查看实名明细 | 打开实名投票结果明细 | 可看到投票人和答案明细 |
| WWAS-09 | 系统 | 保护匿名性 | 处理匿名投票答案 | 不展示投票人与答案映射 |
| WWAS-10 | 系统 | 结束投票 | 到达结束时间或管理员提前结束 | 状态变为已结束，结果对员工可见 |

## 6. 后续迭代建议

| 方向 | 说明 |
|---|---|
| 投票范围控制 | 支持按部门、人员名单、组织层级限制参与范围。 |
| 草稿和预览 | 管理员可保存草稿、预览员工端效果后再发布。 |
| 结果导出 | 支持导出 Excel、图片或 PDF 报告。 |
| 模板复用 | 提供常用投票模板，如满意度、活动选择、评优评奖。 |
| 消息通知 | 投票创建、临近截止、结果发布时发送邮件或企业 IM 通知。 |
| 单点登录 | 接入企业 SSO、LDAP、企业微信或飞书账号体系。 |
| 高级题型 | 支持排序题、矩阵题、NPS、条件跳题。 |
