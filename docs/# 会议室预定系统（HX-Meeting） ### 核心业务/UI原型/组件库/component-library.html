HX-Meeting 移动端 H5 组件库

以手机画布展示移动 H5 组件状态和业务用法，不再以 PC 表格、桌面看板或侧边栏为主。

导航框架

  • MobileTopBar：标题、用户、未读角标
  • BottomNavigation：首页/预约/我的预约/核销/通知
  • QuickActionCard：首页 2 列入口，管理员入口独立卡片

预约组件

  • DateStrip：未来 30 天内日期横向选择
  • RoomCard：位置、容量、占用摘要
  • TimeSlotChip：30 分钟粒度，连续选择，最多 2 小时
  • BookingBottomSheet：底部预约表单

会议与待办

  • MeetingCard：待签到、已签到、违规释放、转让状态
  • TodoCard：待签到/转让/禁用置顶
  • TransferConfirmPanel：责任说明 + 确认/拒绝

签到与反馈

  • CheckInLandingCard：扫码结果、命中会议、确认签到
  • ResultFeedback：成功/失败/恢复动作
  • Toast/InlineAlert：规则提示、冲突、禁用

表单与管理

  • MobileFormItem：单列字段，错误贴近字段
  • PersonPicker：参会人与转让接收人
  • AdminRoomCard：移动端会议室管理卡片

适配规则

  • 点击热区 ≥ 44px
  • 底部导航/抽屉预留安全区
  • 状态必须颜色 + 文案
  • 普通占用详情仅展示姓名和部门
