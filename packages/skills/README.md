# Runtime Skills

`packages/skills/` stores platform runtime skills that are invoked by agents inside the AI Software Factory.

This directory is separate from the top-level `skills/` directory:
- `skills/`: Codex-readable authoring skills
- `packages/skills/`: runtime modules used by the platform
- `apps/server/src/main/java/com/aifactory/skill/`: Spring Boot adapters for runtime skills
